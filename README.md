# knapsack-problem

## Input Format:

```
n K
v_0 w_0
v_1 w_1
... ...
v_n-1 w_n-1
```

* Every problem has input contains n + 1 lines
* n: number of items in the problem
* K: the capacity of the knapsack
* The remaining n lines present the data for each of the items: value & weight
